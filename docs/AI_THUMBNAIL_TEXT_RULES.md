# BELAUG Thumbnail Text Rules (JP-safe workflow)

這份文件用來規範：當封面上包含日文（尤其是漢字）時，**禁止用生成模型「畫字」**，一律改用「移除舊字 + 字型打字」的方式，確保字形正確、無殘影、不傷到背景。

---

## ✅ 核心規則（必遵守）

### 1) 禁止生成文字（JP 漢字不可用 AI 畫）
- **日文漢字/假名不得用生成模型直接生成**（避免筆畫錯、殘影、亂碼、看似正確但其實錯）。

### 2) 先乾淨移除舊字（只遮罩字像素）
- 移除時只能遮罩「文字像素」做修補（inpaint）。
- **禁止整條帶狀遮罩**（會糊背景、留下橫條、污染星星/紋理）。

### 3) 再用真字型打字（Glyph 正確）
- 使用可靠 CJK 字型（例如 Noto Sans CJK / 你指定字型）。
- 需匹配：字重、大小、顏色、對齊、字距。

### 4) 輸出前檢查（必做）
- 200% 放大檢查：
  - 無殘影、無重疊、無亂碼碎片
  - 英文字（如 “Exploration”）**不受影響**
  - 日文空格規則完全正確（例：`星オイジュス` **不得有空格**）
